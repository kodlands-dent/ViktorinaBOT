# ViktorinaBOT
:)
Это мой проект бота-викторины на Python. Бот уже умеет задавать вопросы из базы, но я хочу сделать его намного круче. Вот что я планирую добавить в ближайшее время:

1. Система очков (Score System) 🏆
Сейчас бот просто говорит «верно» или «неверно». Я хочу добавить счетчик:

За каждый правильный ответ пользователь будет получать баллы.

В конце викторины бот будет выводить итоговый результат (например: «Ты набрал 8/10 очков!»).

Техническая часть: Добавлю новую колонку в таблицу пользователей в БД, чтобы очки не пропадали после перезапуска.

2. Огромная база сложных вопросов 🧠
Хочу, чтобы викторину было реально трудно пройти.

Найду и добавлю в SQLite вопросы по истории, науке и программированию, на которые не ответишь без гугла.

Сделаю так, чтобы вопросы выбирались рандомно, а не шли всегда в одном и том же порядке.

3. Разные уровни сложности и «Бонусные вопросы» 💎
Это план на «максималки», если успею по времени:

Обычные вопросы: дают 10 очков.

Сложные вопросы: за них будет даваться сразу 30 или 50 очков.

В базе данных добавлю колонку difficulty, чтобы бот понимал, сколько баллов начислять за конкретный вопрос.
